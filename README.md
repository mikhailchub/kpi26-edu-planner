# EduPlanner - Планувальник навчальних задач

Вебзастосунок для студентів, який допомагає організувати навчальний процес, відстежувати дедлайни та контролювати навантаження.

## Основні можливості

- **Управління задачами**: створення, редагування, зміна статусу та видалення.
- **Розумне відстеження**: автоматичне визначення прострочених задач.
- **Фільтрація та сортування**: зручний пошук за назвою, предметом, пріоритетом або статусом.
- **Статистика**: наочна панель показників вашої продуктивності.
- **Локальність**: всі дані зберігаються у вашому браузері (localStorage).

## Технологічний стек

- **Frontend**: React + TypeScript + Vite.
- **Стан**: Zustand + Persist.
- **Іконки**: Lucide React.
- **Дизайн**: Vanilla CSS (Modern Dark UI).

## Як запустити локально

1. Встановіть залежності:
   ```bash
   npm install
   ```

2. Запустіть сервер розробки:
   ```bash
   npm run dev
   ```

3. Відкрийте застосунок за адресою `http://localhost:5173`.

## Складання та розгортання

Для створення продуктової версії:
```bash
npm run build
```

Застосунок автоматично розгортається на **GitHub Pages** при пуші в гілку `main` через GitHub Actions.

## Структура проекту

- `src/store/`: логіка управління станом та бізнес-правила.
- `src/components/`: інтерфейсні компоненти.
- `src/utils/`: валідація та допоміжні функції.
- `docs/sdr/`: архітектурні рішення (SDR).
- `Requirements.md`: детальні вимоги до системи.
